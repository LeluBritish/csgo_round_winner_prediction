# CS:GO Round Winner Prediction

This project explores whether it is possible to predict the outcome of a Counter-Strike: Global Offensive (CS:GO) round using information available during the match. As a fan of competitive gaming and data science, I wanted to investigate how machine learning can be applied to esports analytics.

The goal of the project is to determine whether the Terrorist or Counter-Terrorist team is likely to win a round based on factors such as team economy, equipment value, player health, and other in-game statistics.

## Dataset

The dataset contains information collected from CS:GO matches, including:

* Team economy
* Equipment value
* Number of players alive
* Player health
* Utility usage
* Match-related statistics

## Project Workflow

The project follows a typical machine learning pipeline:

1. Data exploration and visualization
2. Data cleaning and preprocessing
3. Feature analysis
4. Model training
5. Hyperparameter tuning
6. Performance evaluation

## Results

The trained model was able to identify important gameplay patterns and predict round outcomes with good accuracy. The results demonstrate how machine learning techniques can be applied to competitive gaming environments.
